### cluster

흔하게 사용하는 방법으로 flex-wrap으로 하위 요소를 상위 요소에 가둔다.

<div style={{ outline: '5px solid hotpink', padding: '1rem', display: 'flex', gap: '1rem', flexWrap: 'wrap' }}>
  <div style={{ border: '3px solid #8f9fef', padding: '0.5rem', background: '#0f1f6f' }}>Lorem</div>
  <div style={{ border: '3px solid #8f9fef', padding: '0.5rem', background: '#0f1f6f' }}>Illum</div>
  <div style={{ border: '3px solid #8f9fef', padding: '0.5rem', background: '#0f1f6f' }}>Minima</div>
  <div style={{ border: '3px solid #8f9fef', padding: '0.5rem', background: '#0f1f6f' }}>Blanditiis</div>
  <div style={{ border: '3px solid #8f9fef', padding: '0.5rem', background: '#0f1f6f' }}>Assumenda</div>
  <div style={{ border: '3px solid #8f9fef', padding: '0.5rem', background: '#0f1f6f' }}>Enim</div>
  <div style={{ border: '3px solid #8f9fef', padding: '0.5rem', background: '#0f1f6f' }}>Ullam</div>
  <div style={{ border: '3px solid #8f9fef', padding: '0.5rem', background: '#0f1f6f' }}>Dignissimos</div>
  <div style={{ border: '3px solid #8f9fef', padding: '0.5rem', background: '#0f1f6f' }}>Ab</div>
  <div style={{ border: '3px solid #8f9fef', padding: '0.5rem', background: '#0f1f6f' }}>Quasi</div>
</div>

```html
<div class="cluster">
  <div class="child">Lorem</div>
  <div class="child">Illum</div>
  <div class="child">Minima</div>
  <div class="child">Blanditiis</div>
```

```css
.cluster {
  outline: 5px solid hotpink;
  padding: 1rem;

  display: flex;
  gap: 1rem;
  flex-wrap: wrap;
}

.child {
  border: 3px solid hsl(230, 75%, 75%);

  background: hsl(230, 75%, 25%);
```
