### useState의 `초깃값과 같은 값`으로 setState를 하면 렌더링이 발생하지 않는다.

```javascript
import { useEffect } from 'react';
import { useState } from 'react';

function App() {
  const [count, setCount] = useState(1);

  console.log(count);

  useEffect(() => {
    setCount(1);
  }, [count]);

  return <></>;
}

export default App;
```

- 출력: 1

### useState의 `초깃값과 다른 값`으로 setState를 하면 이전값과 현재값이 같더라도 렌더링이 발생한다.

```javascript
import { useEffect } from 'react';
import { useState } from 'react';

function App() {
  const [count, setCount] = useState(1);

  console.log(count);

  useEffect(() => {
    setCount(2);
  }, [count]);

  return <></>;
}

export default App;
```

- 출력: 1 2 2
- count가 1일 때 1 출력
- count가 1 -> 2로 바뀐 후 리렌더링 2 출력
- `count가 2 -> 2로 같지만 한 번 더 리렌더링 2 출력`
